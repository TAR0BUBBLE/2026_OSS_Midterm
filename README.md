# 세계 수도 챌린지 🌍

Streamlit으로 구현한 **나라별 수도 퀴즈 웹 애플리케이션**입니다.  




## 1. 프로젝트 소개

이번 프로젝트는 오픈소스프로그래밍 중간 대체 과제를 위해 제작한 Streamlit 기반 퀴즈 앱입니다.



앱 이름은 **세계 수도 챌린지**이며,  
세계 여러 나라의 수도를 맞히는 과정을 통해 지리 상식을 점검할 수 있도록 구성했습니다.



## 2. 주요 기능

### 1) 첫 화면에 학번 / 이름 표시
앱 실행 시 첫 화면에서 제출자의 학번과 이름을 확인할 수 있도록 구성했습니다.

### 2) 로그인 기능
미리 정의된 사용자 정보를 JSON 파일로 분리해 두고,  
입력한 아이디와 비밀번호를 비교하여 로그인 성공/실패를 처리합니다.

- 사용자 입력
- 로그인 성공 / 실패 처리
- 로그인 상태 구분

을 모두 구현했습니다.

### 3) 캐싱 기능
퀴즈 데이터와 사용자 정보를 불러오는 과정에서 Streamlit의 캐싱 기능을 활용했습니다.

퀴즈 데이터는 JSON 파일에서 읽어오는데,  
앱이 다시 실행될 때마다 같은 파일을 반복적으로 읽지 않도록 캐싱을 적용해  
불필요한 로딩을 줄이고 반응 속도를 높였습니다.

### 4) 퀴즈 기능
사용자는 대륙을 선택한 뒤 수도 퀴즈를 시작할 수 있습니다.

- 문제 제시
- 사용자 응답 입력
- 정답 여부 판별
- 점수 계산
- 최종 결과 확인

흐름이 모두 구현되어 있습니다.

### 5) 국기 이미지 표시
각 문제에서 해당 국가의 국기를 함께 보여주어,  
단순 텍스트 문제보다 더 직관적이고 흥미롭게 퀴즈를 풀 수 있도록 했습니다.

### 6) 결과 등급 제공
정답 개수에 따라 아래와 같은 등급이 부여됩니다.

- 여행 초보
- 지리 탐험가
- 수도 마스터



## 3. 프로젝트 구조

```bash
world-capital-challenge/
├── app.py
├── requirements.txt
├── README.md
├── .gitignore
├── data/
│   ├── capitals_quiz.json
│   └── users.json
└── assets/
    ├── header-map.jpg
    └── flags/
```

- app.py : 메인 Streamlit 실행 파일
- data/capitals_quiz.json : 퀴즈 문제 데이터
- data/users.json : 로그인용 사용자 정보
- assets/header-map.jpg : 메인 헤더 배경 이미지
- assets/flags/ : 국가별 국기 이미지 파일
- requirements.txt : 실행 환경 재현용 패키지 목록

## 4. 실행방법
1) 가상환경 생성
```bash
python3 -m venv venv
source venv/bin/activate
```

2) 패키지 설치
```bash
pip install -r requirements.txt
```

3) 앱 실행
```bash
streamlit run app.py
```
